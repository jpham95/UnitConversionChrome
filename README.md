# Unit Conversion Extension for Chrome
 
Converts highlighted units to their imperial/metric opposite and displays in a tooltip.
Currently supports:
  - °C <-> °F
  - kilometers <-> miles
  - kilograms <-> pounds
  - centimeters <-> inches
  - grams <-> ounces
